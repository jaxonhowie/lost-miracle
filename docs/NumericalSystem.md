## 一、角色属性系统

### 1.1 基础属性 (Primary Attributes)

| 属性 | 缩写 | 说明 |
|------|------|------|
| 力量 | STR | 影响生命值、近战攻击力、暴击率 |
| 敏捷 | AGI | 影响防御力、攻速、闪避率、远程攻击 |
| 智力 | INT | 影响魔法值、魔法攻击、魔法防御 |

**升级奖励**：
- 每级自动 +1 **主属性**
- 每3级循环额外 +1 属性：**副属性A → 副属性B → 主属性**
- 无可分配属性点，完全自动分配

**各职业属性成长规则**：

| 职业 | 主属性 | 副属性A | 副属性B | 初始值 |
|------|--------|---------|---------|--------|
| 战士 Warrior | STR | AGI | INT | STR=10, AGI=3, INT=3 |
| 游侠 Ranger | AGI | STR | INT | STR=3, AGI=10, INT=3 |
| 刺客 Assassin | AGI | STR | INT | STR=3, AGI=10, INT=3 |
| 精灵 Elven | INT | STR | AGI | STR=3, AGI=3, INT=10 |

**升级循环示例（战士）**：

| 等级 | 自动获得 | 说明 |
|------|----------|------|
| Lv1→2 | +1 STR, +1 AGI | 主属性+副属性A |
| Lv2→3 | +1 STR, +1 INT | 主属性+副属性B |
| Lv3→4 | +1 STR, +1 STR | 主属性+主属性 |
| Lv4→5 | +1 STR, +1 AGI | 主属性+副属性A |
| Lv5→6 | +1 STR, +1 INT | 主属性+副属性B |
| Lv6→7 | +1 STR, +1 STR | 主属性+主属性 |
| ... | ... | 每3级循环 |

**通用公式**：
```
每级获得：+1 主属性
额外获得（每3级循环）：
  (level - 1) % 3 == 0 → +1 副属性A
  (level - 1) % 3 == 1 → +1 副属性B
  (level - 1) % 3 == 2 → +1 主属性
```

> 注：无手动分配机制，升级后属性自动增长

## 三、战斗公式系统

### 3.1 伤害计算

```
基础伤害 = max(1, 攻击力 - 防御力 × 0.5)

暴击判定：
if random(0, 1) < crit_rate:
    基础伤害 = 基础伤害 × crit_dmg

闪避判定：
if random(0, 1) < target.dodge:
    基础伤害 = 0

技能伤害：
技能伤害 = 基础伤害 × 技能倍率 × (1 + skill_damage)

最终伤害 = floor(基础伤害 × (1 + 亡灵增伤 + Boss增伤) × (1 - target.damage_reduce))
```

### 3.2 吸血计算

```
吸血回复 = 最终伤害 × lifesteal
```

### 3.3 等级差修正

| 等级差 | 命中率修正 | 暴击率修正 | 伤害修正 |
|--------|-----------|-----------|----------|
| -5 或更低 | -15% | -5% | -20% |
| -4 | -12% | -4% | -16% |
| -3 | -9% | -3% | -12% |
| -2 | -6% | -2% | -8% |
| -1 | -3% | -1% | -4% |
| 0 | 0% | 0% | 0% |
| +1 | +2% | +1% | +2% |
| +2 | +4% | +2% | +4% |
| +3 | +6% | +3% | +6% |
| +4 | +8% | +4% | +8% |
| +5 或更高 | +10% | +5% | +10% |

## 十二、数值调整指南

### 12.1 如果战斗太难

1. 降低怪物ATK/DEF
2. 提高玩家基础HP
3. 增加掉落回复道具

### 12.2 如果战斗太简单

1. 提高怪物HP
2. 增加怪物技能使用频率
3. 降低掉落品质概率

### 12.3 如果成长太慢

1. 提高经验产出
2. 降低升级经验需求
3. 增加装备掉落率

### 12.4 如果成长太快

1. 降低经验产出
2. 提高升级经验需求
3. 降低装备掉落率
