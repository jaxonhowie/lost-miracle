# 装备系统 — R2（Reign of Revolution）参考设计

> 参考韩国 MMORPG **R2 Online** 的装备成长体系，适配本项目「失落奇迹：亡灵地牢」。

---

## 一、R2 核心机制对照

| R2 原版概念 | 本项目实现 | 说明 |
|------------|-----------|------|
| 强化卷轴 (+N) | 强化石 / 受祝福强化石 | 消耗卷轴提升 +N |
| 安全强化区 (+0~+1 防具等) | `safe_enhance_until` | 默认 +3 前失败不损毁 |
| 失败损毁装备 | `break_on_fail` | +4 起普通卷轴失败有概率损毁 |
| 祝福装备 | `is_blessed` | 失败时 50% 概率保留装备 |
| 祝福卷轴 | 受祝福强化石 | 更高成功率，失败不损毁 |
| 装备品阶 (白/绿/蓝/金) | `grade` | 普通/高级/稀有/独特 |
| 制作合成 | 后续扩展 | 材料 + 底材 → 固定属性装 |
| 亡灵强化 (Slain) | 套装 `undead_damage` | 仅对怪物生效（已实现） |
| 随机附加能力 | `affixes` | 精英/Boss 掉落随机词条 |

## 二、装备实例结构

```json
{
  "uid": "eq_xxx",
  "base_id": "bone_axe",
  "name": "骸骨战斧",
  "slot": "weapon",
  "grade": "rare",
  "enhance_level": 7,
  "is_blessed": false,
  "safe_enhance_until": 3,
  "base_stats": {"atk": 35},
  "affixes": [
    {"id": "crit_rate", "stat": "crit_rate", "value": 0.05, "name": "暴击率"}
  ],
  "set_id": "undead_hunter",
  "effects": {"tier1": {}, "tier2": {}, "tier3": {}}
}
```

### 字段说明

| 字段 | 说明 |
|------|------|
| `grade` | 品阶：normal / advanced / rare / unique |
| `enhance_level` | 强化 +N（R2 核心成长） |
| `is_blessed` | 祝福装备，失败损毁时 50% 保留 |
| `safe_enhance_until` | 安全强化上限（含），默认 3 |
| `affixes` | 随机附加词条（掉落时生成） |

## 三、品阶系统（R2 底子稀有度）

| 品阶 | 中文 | 颜色 | 属性倍率 | 掉落来源 |
|------|------|------|----------|----------|
| normal | 普通 | 白 | 1.00× | 普通怪 |
| advanced | 高级 | 绿 | 1.08× | 普通怪小概率 / 宝箱 |
| rare | 稀有 | 蓝 | 1.18× | 精英怪 |
| unique | 独特 | 金 | 1.30× | Boss |

> 视觉品质（白/蓝/紫/橙）仍由 **强化 +N** 决定，与 R2 中「+N 越高越强」一致。

## 四、强化规则（R2 卷轴体系）

### 4.1 成功率

| +N | 普通强化石 | 受祝福强化石 |
|----|-----------|-------------|
| +0→+1 | 100% | 100% |
| +1→+2 | 100% | 100% |
| +2→+3 | 100% | 100% |
| +3→+4 | 30% | 100% |
| +4→+5 | 28% | 33% |
| +5→+6 | 20% | 25% |
| +6→+7 | 18% | 23% |
| +7→+8 | 15% | 20% |
| +8→+9 | 13% | 18% |
| +9→+10 | 10% | 15% |

### 4.2 失败惩罚（R2 特色）

| 条件 | 材料消耗 | 装备损毁 |
|------|----------|----------|
| +N ≤ 安全区 (默认+3) | 是 | **否** |
| +N > 安全区 + 普通石失败 | 是 | **35% 概率损毁** |
| +N > 安全区 + 祝福石失败 | 是 | **否** |
| 祝福装备 + 将损毁 | — | **50% 概率保留** |

### 4.3 强化属性成长（参考 R2）

- **武器**：每级 +1 ATK（+5 额外 +2，+6~+10 每级 +3）
- **防具**：每级 +1 DEF/HP（+6 起 +2，+8 起 +3）
- **饰品**：混合成长

## 七、制作合成（规划）

参考 R2「制作」窗口 (快捷键 J)：

```
底材装备 + 合成剂 + 宝石 → 高级装备（固定词条）
```

示例（规划）：
- 铁剑 + 一般合成剂 + 红宝石 → 烈焰短剑（固定 +火伤）
- 骸骨护手 + 高级合成剂 + 纳克里斯特尔 → 亡灵拳套（亡灵 Slain Lv1）

**当前版本未实现 UI**，数据结构已预留 `craft_recipes.json` 扩展位。

## 八、与旧系统差异

| 旧设计 | R2 设计 |
|--------|---------|
| 强化失败只耗材料 | +4 起可能损毁装备 |
| 无装备品阶 | 普通/高级/稀有/独特 |
| 无随机词条 | 精英/Boss 掉词条 |
| 品质仅看 +N | +N 视觉品质 + 底子品阶并存 |
